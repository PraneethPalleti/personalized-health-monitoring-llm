# Personalized Health Monitoring Using Machine Learning and LLMs

## Overview

This project analyzes wearable health data to detect unusual physiological behavior and generate AI-powered health insights.

The system combines feature engineering, personalized baseline modeling, Isolation Forest anomaly detection, event detection, and LLM-based event interpretation.

## Features

* Personalized heart-rate baseline generation
* Heart-rate variability analysis
* Isolation Forest anomaly detection
* Event detection and severity classification
* LLM-powered event explanations
* Health intelligence reporting

## Technologies

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Groq API
* Llama 3.3 70B

## Workflow

Wearable Data → Feature Engineering → Anomaly Detection → Event Detection → LLM Explanations → Health Report
